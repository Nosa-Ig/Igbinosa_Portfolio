# 🏥 Healthcare Insurance Cost Prediction
🔗 Project Link  
https://github.com/Nosa-Ig/Igbinosa_Portfolio/tree/main/Healthcare_Insurance_Project

### 📊 Project Overview
- Built predictive models to estimate **hospitalisation costs**
- Used **Gradient Boosting and Ridge Regression**
- Applied **Stratified 5-Fold Cross Validation**
- Performed feature engineering including:
  - Age calculation
  - Gender extraction from names
  - Medical risk transformations
- Identified key cost drivers:
  - Smoking
  - BMI
  - Hospital Tier
  - Major Surgeries

### 📈 Business Impact
Enabled **data-driven insurance pricing decisions** through predictive analytics and interactive dashboards.

### ⚙️ Technical Stack

- Python (Google Colab)
- Pandas & NumPy
- Scikit-learn
- SciPy statistical testing
- MySQL
- Tableau dashboards
- Matplotlib visualisations

# 📦 Sales Forecasting & Warehouse Optimisation

🔗 Project Link  
https://github.com/Nosa-Ig/Igbinosa_Portfolio/tree/main/Sales%20Forecasting%20and%20Warehouse%20Optimisation%20System%20using%20Deep%20Learning

### 📊 Project Overview

Developed a **deep learning forecasting system** to predict retail demand and optimise warehouse distribution.

### 🤖 Machine Learning Architecture

- Conv1D + BiLSTM hybrid neural network
- Multi-horizon forecasting (4 weeks ahead)
- 1.5M+ time series training windows
- Categorical embeddings for product & store behaviour

### ☁️ Cloud Architecture

- Google BigQuery for data warehousing
- Vertex AI for scalable training
- NVIDIA T4 GPU acceleration
- Production-ready evaluation metrics (RMSE, WMAPE)

### ⚙️ Technical Stack

- Python (Pandas, NumPy)
- TensorFlow & Keras
- Google Cloud Platform
- BigQuery ML
- GPU acceleration
